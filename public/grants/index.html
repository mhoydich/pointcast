Grant Request Strategy

PointCast.xyz is a compact, weirdly useful media lab for AI-native publishing, social objects, onchain culture, public-good experiments, and fast local prototypes. The strategy: submit many small, specific, milestone-based requests instead of one vague mega-pitch.

Approach

Package PointCast as grant-native public infrastructure

Lead with what funders can repeat back: PointCast turns AI agents, feeds, generative media, and onchain objects into tiny broadcast channels that people can inspect, remix, collect, and fund.

Translate the same core work per ecosystem

OpenAI and Anthropic get the AI workflow angle. Tezos, Ethereum, Solana, and Zora get creator tooling, culture, protocol adoption, and open-source experiments. YC gets the company story.

Ask for concrete milestones

Every request should buy a visible artifact: a public demo, open-source repo, creator pilot, documentation pack, user study, or onchain collection with measured distribution.

Reusable Application Packet

Core paragraph

PointCast.xyz builds AI-native broadcast channels for creators, communities, and protocol ecosystems. It combines agent-assisted production, lightweight web publishing, social distribution, and optional onchain artifacts so a single builder can ship useful cultural software at unusual speed.

Budget frame

Funding covers model/API usage, design and engineering time, public documentation, pilot participants, hosting, smart-contract experiments, and post-grant reporting.

Proof to attach

Live demos, GitHub links, screenshots, traffic or usage notes, creator collaborations, onchain contracts, and a short founder note from Michael Hoydich.
