Claim Rules

Do the task

Use the linked PointCast path. Collect, enter a room, test, remix, play, write, or submit a concrete improvement.

Attach proof

Proof can be a URL, wallet address, screenshot, block ID, room trace, GitHub link, or short note explaining what changed.

Claim token credit

Rewards start as public accounting. When the token rail is live, valid claims can be settled as POINT, DRUM, Tezos, or ecosystem-specific credits.

Reward Notes

Anti-spam

Rewards are capped per wallet and task until the claim system is formalized. Repeated low-effort actions do not compound automatically.

Grant compatible

Each grant can fund a specific reward pool: art collectors, room testers, developer bounties, AI workflow reports, or public documentation.

Human review

Michael can approve, reject, or resize claims. The public task board is the source of truth until automated settlement exists.
