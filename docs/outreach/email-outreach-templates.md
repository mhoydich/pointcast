# Email outreach templates

Three base templates. Keep emails under 150 words. No attachments. No
follow-up within 10 days. Track in a lightweight CRM (a spreadsheet
works).

---

## Template A — newsletter curator (TLDR AI / Ben's Bites / Bytes)

Use for: high-volume curators who need a 1-sentence hook + a link.

```
Subject: llms.txt + stripped-HTML for AI crawlers — case study

Hi {first_name},

I built a site (pointcast.xyz) that ships llms.txt, agents.json,
Content-Signals opt-in, AND a Cloudflare Pages middleware that serves
~12% lighter HTML to AI crawlers. All open, CC0-flavored, with live
code links.

It's the most complete agent-native reference site I've seen — might
fit {newsletter_name}'s readership that's building in the same lane.

Canonical writeup: https://pointcast.xyz/agent-native

If there's a better way to pitch {newsletter_name}, let me know.

Mike Hoydich
pointcast.xyz · El Segundo, CA
```

## Pitching rules

1. **Never send more than one follow-up.** Wait 10 days. If no response,
   they're not interested; move on.

2. **Reference their actual work.** If you can't name a specific post
   or episode you read, don't pitch them.

3. **No attachments.** Use links.

4. **Keep under 150 words.** Every word above that reduces reply rate.

5. **Put the canonical URL in the first paragraph.** People decide
   to click or not in the first 5 seconds.

6. **Don't send generic mass-pitches.** Individualize each one. 20
   targeted pitches will out-perform 200 spray-and-pray every time.

7. **Track outcomes.** If a particular template isn't landing, iterate
   on the subject line and hook paragraph first.
