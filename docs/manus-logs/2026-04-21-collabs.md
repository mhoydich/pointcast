# Manus collab paths log - 2026-04-21

## Summary

This branch turns the collaboration idea into concrete public doors:

- `/collabs` publishes the collaborator registry, federation spec, guest-block path, local-station path, node path, compute path, polls path, and quiet-watching path.
- `/collabs.json` mirrors the registry and federation steps for agents.
- `/polls` includes the seven-option "How would you contribute to PointCast?" poll.
- `docs/federation-examples/` stages a drop-in `/compute.json` example for the first external federation peer.

## Seven paths

1. Run a node or broadcast an agent.
2. Write a guest block.
3. Host a local meetup or station.
4. Federate your own site.
5. Seed polls and ask the room.
6. Donate compute or GPU hours.
7. Just watch, read, and stay curious.
