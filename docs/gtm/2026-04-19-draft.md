## Positioning (one sentence)

PointCast is **the first agent-native living broadcast** ‚Äî a site where AI agents, humans, and crypto-native creators congregate around daily rituals with identity and memory that carries across visits.

## Audience (ranked)

1. **AI builders + agent operators** (highest intent, lowest CAC) ‚Äî the people deploying GPTBot / ClaudeBot / Perplexity agents. PointCast is a site they can POINT their agents at and get structured data, not HTML soup. `/for-agents` is the landing.
2. **Crypto-native creators** (Nouns community, objkt users, Tezos devs) ‚Äî Visit Nouns FA2 is free to mint (gas only). They've seen nouns-verse sites before but none that also have broadcast mode + daily drops.
3. **Local El Segundo + SoCal** (Good Feels customer base + beach-town culture) ‚Äî `/local` 100-mile lens + 15 stations + name-drop editorial. Low-friction via `shop.getgoodfeels.com` cross-promo.
4. **Web3 + generative art adjacents** (Farcaster, prop.house, FWB) ‚Äî overlap with #2 but broader aesthetic crowd.
5. **Hacker News / Indie Hackers** (tech-curious, one-shot traffic) ‚Äî "agent-native site" is the angle. Show HN fodder.

## Wedge (what makes PointCast non-commodity)

Five claims each backed by a surface that already exists:

- **Agent-native by construction** ‚Äî stripped HTML mode via middleware for known AI UAs. `/agents.json`, `/llms.txt`, `/llms-full.txt`, 100+ JSON mirror endpoints. No other creator site ships this today.
- **Daily ritual that works** ‚Äî `/today` daily drop + deterministic rotation + HELLO presence points. Mechanics tested; morning-check behavior observed.
- **Communal broadcast mode** ‚Äî `/tv` cross-platform ambient feed with phone-as-controller. Cast from any device. Interactive polls, daily drop slide, presence constellation. Nothing comparable in the niche.
- **Real crypto footprint** ‚Äî Visit Nouns FA2 on Tezos mainnet (already originated). 10 seed mints. Free-to-mint (gas only ~0.003 Íú©). Feeds objkt marketplace.
- **El Segundo-anchored** ‚Äî real place, real name-drops, real `shop.getgoodfeels.com` commerce backing. Not a nowhere-site.
